(task_1_screen.jpg)

- в первой колонке выводить номер по порядку, начиная с 1
- во второй колонке выводить  произведение 2-х чисел (номера соответствуюшей строки)

/* =================== как выполнять задание =================== */
- код таблицы сохранять как текст в переменную, например по имени  res
- по окончанию цикла переменную вывести в 
          <div id="result_1"></div> 
  используя
         document.getElementById('result_1').innerHTML = res;
Эта конструкция заменит пример кода таблицы на тот, который будет
в переменной res
